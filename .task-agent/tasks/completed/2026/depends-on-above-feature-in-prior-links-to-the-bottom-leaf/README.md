# depends-on-above feature in prior links to the bottom leaf

depends on above should link to the root of the above tree, if there is a tree. It should not link to the bottom leaf of the tree. Once it is in the tree, it should be able to move up and down the top level children and be able to be made a subtask of the top-level child immediately above it. That should provide a precise, fine grained linking functionality.
